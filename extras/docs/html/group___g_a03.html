◆ setup()

void KT0915::setup ( int  reset_pin,
uint8_t  crystal_type = CRYSTAL_32KHZ,
uint8_t  ref_clock = REF_CLOCK_DISABLE 
)

Receiver startup.

Use this method to define the MCU (Arduino) RESET pin and the crystal type you are using.

The options for the crystal can be seen in the table below.

If you omit the crystal type parameter, will be considered 0 (32.768KHz).

The code below shows how to use the setup function.

#include <KT0915.h>
#define RESET_PIN 12 // set it to -1 if you want to use the RST pin of your MCU.
KT0915 radio;
void setup() {
// Set RESET_PIN to -1 if you are using the Arduino RST pin; Select CRYSTAL_32KHZ, CRYSTAL_12MHZ etc
// radio.setup(RESET_PIN); Instead the line below, if you use this line, the crystal type considered will be 32.768KHz.
radio.setup(RESET_PIN, CRYSTAL_12MHZ, REF_CLOCK_DISABLE );
}
